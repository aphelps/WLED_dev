## General
* Limit lines to 80 characters when practical, but never over 120 characters
* Indentation uses spaces, never tabs
* Variable names should be meaningful

## C/C++
* 2-character indentation
* Variable names no more than 16 characters, lowercase.  Counter variables can 
  be single character
* #define, goto, etc names use uppercase
* "char *ptr" not "char* ptr"
* Space between keyword and expression, ie "if (stuff)", "sizeof (variable)"
* All if, switch, while, etc statements use {} even if not required
* newline between return type and function name
* { at end of function definition

## Python
* 4-character indentation
* Use PEP8 unless otherwise indicated
